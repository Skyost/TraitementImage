Traitement d'images
==================

Traitement d'image en Java.

## Fonctionnalités
* Enregistrement et chargement de matrices de convolution (*.mtx).
* Matrices prédéfinnies.
* Zoom avec filtre d'antialiasing.
* Enregistrement des prévisualisations.
